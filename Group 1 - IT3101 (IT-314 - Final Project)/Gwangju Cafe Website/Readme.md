Gwangju Café Website

Final Project — IT3101 / IT-314

Overview:
    The Gwangju Café Website is a web-based application developed as part of the IT3101 (IT-314) Final Project.
    It features an interactive café menu, categorized product pages, and user account features such as a sign-up page.
    The site is designed to be visually appealing, easy to navigate, and user-friendly—providing a realistic café-style browsing experience.

Project Structure:
    Main Page: SignUp.html — User registration page
    Additional pages: Product Categories & Pages

The website includes multiple item categories, each with sub-pages:
    Category 1 — Coffee
        Iced Coffee
        Hot Coffee
        Coffee Overview

    Category 2 — Non-Coffee Drinks
        Latte
        Fruit Tea
        Non-Coffee Overview

    Category 3 — Secret Menu
        Soda Series
        Fusion Series
        Biscoff Series
        Secret Menu Overview
        
    Category 4 — Pastries
        Cinnamon Roll
        Cake
        Pastries Overview

    Category 5 — Takoyaki
        Original Takoyaki
        Spicy Takoyaki

    Category 6 — Ramen
        Beef Ramen
        Chicken Ramen

    Extract the ZIP file.
